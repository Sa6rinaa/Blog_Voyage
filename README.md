# ✈️ Azurea - Agence de Voyage Premium

Azurea est une plateforme moderne de réservation de voyages. Ce projet transforme un simple blog en une interface utilisateur (UI) élégante et fonctionnelle permettant aux utilisateurs de découvrir des destinations de rêve, de s'inscrire et de simuler des réservations.

## ✨ Fonctionnalités
- **Design Moderne** : Interface épurée avec flou d'arrière-plan (backdrop-filter) et cartes dynamiques.
- **Système d'Authentification** : Inscription et connexion sécurisées.
- **Expérience Utilisateur (UX)** : Redirection intelligente vers la réservation après connexion.
- **Responsive** : S'adapte parfaitement aux mobiles et tablettes.

## 🛠️ Technologies Utilisées
- **Frontend** : HTML5, CSS3 (Variables, Flexbox, Grid).
- **Backend** : PHP 8.x.
- **Gestion de Session** : PHP Sessions pour maintenir l'utilisateur connecté.
